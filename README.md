# ⏳ HourTracker - מעקב שעות עבודה ושכר

![Project Status](https://img.shields.io/badge/Status-Live-success)
![Build](https://img.shields.io/badge/Build-Spring%20Boot%20%26%20React-blue)

HourTracker היא מערכת למעקב אחר משמרות וחישוב משכורת אשר נוצרה בעבור מקום העבודה הקודם שלי - מחלקת הביטחון של מלון ממילא.
הפרויקט נבנה כפתרון Full Stack, עם דגש על ממשק משתמש נקי וחישוב שכר מדויק בזמן אמת.

🔗 **[לצפייה באתר (Live Demo)]([https://hourtracker.onrender.com](https://hourtracker-nxr7.onrender.com))**

---

## 🚀 פיצ'רים מרכזיים

* ✅ **שעון נוכחות:** כניסה ויציאה (Clock In / Out) בלחיצת כפתור.
* 💰 **חישוב שכר:** הגדרת תעריף שעתי וצפייה ברווח היומי והחודשי בזמן אמת.
* 📅 **היסטוריית משמרות:** צפייה ברשימת המשמרות לפי חודשים.
* 📊 **סטטיסטיקות:** סיכום שעות חודשי ושכר כולל.
* 🔐 **אימות משתמשים:** כניסה מאובטחת באמצעות Google OAuth2.
* 🌙 **עיצוב מודרני:** ממשק רספונסיבי התומך במובייל).

---

## 🛠️ טכנולוגיות (Tech Stack)

הפרויקט בנוי בארכיטקטורת Client-Server קלאסית:

### **Backend (צד שרת)**
* **Java / Spring Boot** - ליבת המערכת.
* **Spring Security & OAuth2** - ניהול משתמשים ואבטחה.
* **MySQL** - מסד נתונים (Production).

### **Frontend (צד לקוח)**
* **React.js** - ספריית UI.
* **Tailwind CSS** - עיצוב וסטיילינג.
* **Axios** - תקשורת עם השרת.
